A single cycle processor made as an assignment for COA course - supports add, sub, and, or, sll, slt,sltu, xor, srl, sra, addi, slti, sltui, xori, ori, andi,slli, srli, srai, sw, lw, beq , bne, blt , bge , jal, jalr instructions
The instructions are loaded via a .dat file - into the byte addressed memory (little endian)
